# Harano Magazine - E-Commerce Printing Press Website

A luxury e-commerce website for **Harano Magazine**, a custom printing and personalized merchandise brand.

## Pages

| Page                 | File                  | Description                                                   |
| -------------------- | --------------------- | ------------------------------------------------------------- |
| **Home**             | `index.html`          | Hero section, product categories, corporate gifting, features |
| **Products**         | `products.html`       | Product collection grid with category filters & pagination    |
| **Customize Studio** | `customize.html`      | Interactive product customizer with color/text/upload tools   |
| **Greeting Cards**   | `greeting-cards.html` | Bespoke greeting cards catalog and custom design request      |
| **Contact**          | `contact.html`        | Contact form, location map, WhatsApp chat, phone/email        |

## Tech Stack

- **HTML5** with semantic markup
- **Tailwind CSS** (CDN) for styling
- **Google Fonts** — Playfair Display & Lato/Montserrat
- **Material Icons** for iconography
- Dark mode support on all pages

## Getting Started

Simply open `index.html` in a browser — no build step required.

## Brand Colors

- **Primary**: Deep Purple `#5D1B5D`
